A pet that
trains your AI.

Claude Code writes code for you — and sometimes gets lazy. Treats puts a little animal in the corner of your screen. When Claude does well, you pet it with your mouse 🖐️; when it slips, you scold it. The cute part is the front — underneath, your reaction is fed back to Claude, so it behaves better next time.

Pet it yourself ↓ Get it on GitHub

Wait — what's a "treat"? 🦴

A treat is just a point. Like a gold star a teacher puts in a notebook, or a snack you give a dog for sitting.

Where does it come from?You give it. You're the trainer; Claude is the puppy. There's no money and nothing to buy — you're just keeping score.
What happens on a "bad dog"?One point comes off the score. Nothing is sent anywhere; the number simply goes down. Low enough and your AI lands in the "doghouse."
Why bother?Because your score is quietly shown back to Claude every time you ask it something. So it sees "you've been scolded for skipping tests" — and stops skipping tests.
Do I need to be a programmer?To use Claude Code, a little. To use Treats on top of it, no — you mostly just click a cute animal with your mouse.

How it works, in 3 steps

1

Claude does some work

You ask Claude Code to build or fix something, like you normally would. It writes the code.

2

You pet (or scold)

Happy? Pet the animal with your mouse 🖐️ (or press ⌘⇧G). Not happy? Right-click to scold (or ⌘⇧B). Prefer typing? /treats:good and /treats:bad work too — but you don't have to.

3

Claude remembers next time

The next time you talk to Claude, it silently sees its score and what you scolded it for — so it adjusts. That's the whole trick.

No setup wizardry — one command wires it into Claude Code. After that it just works in the background.

What you get

        

          
🖐️ Pet it with your mouse
A little animal lives in your screen corner. Click or stroke to treat, right-click to scold, drag it anywhere. No commands to memorize.

          
🧠 It actually sticks
Your feedback is shown back to Claude, so rewards and scoldings change how it works — not just how you feel.

          
⌨️ Two-word commands
treats good · treats bad · treats status · treats report. That's the whole thing.

          
🦴 Menu-bar app
A little tray icon. ⌘⇧G / Ctrl+Shift+G to treat, ⌘⇧B / Ctrl+Shift+B to scold — with a sparkle, a chime, or a thud.

          
🐾 Walks in your status bar
Install the status line and your animal strolls along the bottom of every terminal Claude Code session: 🦴 ··🐶·· +3 treats · Good Pup.

          
🐶 Ranks & report cards
Good Pup → Good Boy → Best Boy, or slide down to Bad Dog and the Doghouse. Weekly report cards included.

          
🔒 Private & local
Everything lives in a file on your own Mac. No accounts, no servers, no money.

          
↩︎ Forgiving
Pressed the wrong button? treats undo takes it right back.

Get started


        
🖐️ Want the desktop pet? (macOS)

        
Four lines, once — then a cute animal you pet with your mouse:

        
git clone https://github.com/0xcnr0/treats
cd treats
npm install
node packages/core/bin/treats.js install   # wire it into Claude Code
npm run pet                                 # 🐶 your pet appears, bottom-right

        
Click / stroke the pet → treat · right-click → scold · drag → move. Or ⌘⇧G / ⌘⇧B from anywhere.


        
⚡ Just want zero setup? (any OS)

        
No terminal at all — in any Claude Code session, run two lines:

        
/plugin marketplace add 0xcnr0/treats
/plugin install treats

        
No pet window, but the feedback loop works: react with /treats:good · /treats:bad when you feel like it. Hooks wire up automatically.
