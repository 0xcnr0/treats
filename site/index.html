Claude Code is an AI assistant that writes code for you in the terminal. Sometimes it nails it. Sometimes it gets lazy. Treats lets you reward the good and correct the bad — give it a treat 🦴 for great work, a "bad dog" 🚫 when it slips. And here's the part that matters: it remembers, and it behaves better next time.

Try the demo ↓ Get it on GitHub

Wait — what's a "treat"? 🦴

A treat is just a point. Like a gold star a teacher puts in a notebook, or a snack you give a dog for sitting.

Where does it come from?You give it. You're the trainer; Claude is the puppy. There's no money and nothing to buy — you're just keeping score.
What happens on a "bad dog"?One point comes off the score. Nothing is sent anywhere; the number simply goes down. Low enough and your AI lands in the "doghouse."
Why bother?Because your score is quietly shown back to Claude every time you ask it something. So it sees "you've been scolded for skipping tests" — and stops skipping tests.
Do I need to be a programmer?To use Claude Code, a little. To use Treats on top of it, no — it's two words in the terminal, or two buttons in the menu bar.

How it works, in 3 steps

1

Claude does some work

You ask Claude Code to build or fix something, like you normally would. It writes the code.

2

You react

Happy with it? treats good → a treat. Not happy? treats bad "no tests" → a scolding. Or use the menu-bar shortcuts: ⌘⇧G / ⌘⇧B on Mac, Ctrl+Shift+G / Ctrl+Shift+B on Windows & Linux.

3

Claude remembers next time

The next time you talk to Claude, it silently sees its score and what you scolded it for — so it adjusts. That's the whole trick.

No setup wizardry — one command wires it into Claude Code. After that it just works in the background.

What you get

        

          
🧠 It actually sticks
Your feedback is shown back to Claude, so rewards and scoldings change how it works — not just how you feel.

          
⌨️ Two-word commands
treats good · treats bad · treats status · treats report. That's the whole thing.

          
🦴 Menu-bar app
A little tray icon. ⌘⇧G / Ctrl+Shift+G to treat, ⌘⇧B / Ctrl+Shift+B to scold — with a sparkle, a chime, or a thud.

Get started

        
You'll need Claude Code and Node.js. The CLI & hooks run on macOS, Windows and Linux; the menu-bar app is best on macOS. Then:

        
git clone https://github.com/0xcnr0/treats
cd treats
npm install

# teach Claude Code to listen (it backs up your settings first)
node packages/core/bin/treats.js install-hooks

# pick your animal (dog · cat · dragon · horse · hamster · parrot)
node packages/core/bin/treats.js animal cat

# now train it
treats good "wrote thorough tests"
treats bad  "skipped the edge cases"

# optional: the menu-bar app (⌘⇧G / Ctrl+Shift+G to treat)
npm run overlay

        
Start a fresh Claude Code session and ask it how many treats it has. It'll know.
